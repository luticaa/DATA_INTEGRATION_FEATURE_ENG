OBJETIVO: adquirir conhecimentos práticos do processo de preparação de dados ponta-a-ponta para aprendizado de máquina.

DETALHAMENTO:

Deve-se escolher conjuntos de dados para subsidiar a construção de um modelo baseado em aprendizado supervisionado. Espera-se que mais de um conjunto de dados original seja integrado para a construção do modelo. Sugere-se que sejam utilizados conjuntos de dados do trabalho anterior, na medida do possível.

É preciso definir o objetivo do modelo, identificando a tarefa preditiva a ser implementada. Em seguida, é preciso definir a variável alvo e implementar os processamentos necessários para produzir o valor do target no formato desejado, construindo um conjunto inicial de instâncias. Na sequência, devem ser produzidos atributos descritivos que possam contribuir para o processo de inferência, concatenando-os às instâncias. Neste processo, é necessário realizar tarefas de limpeza de dados, incluindo, necessariamente, identificação de erros e valores faltantes, reparação ou imputação de dados, além de escala e normalização, se o algoritmo de ML escolhido exigir. Se os conjuntos de dados de entrada não tiverem erros ou ruídos, é preciso inserir artificialmente erros e ruídos para posterior limpeza.

Por fim, treinar e avaliar modelos, aplicando processos típicos de ML, como validação cruzada, seleção de features, tuning de hiperparâmetros, etc. Procure aprimorar a capacidade preditiva acrescentando features derivados ou complementares, indicando as tarefas de preparação desses features e a acurácia do modelo antes e depois da inclusão dos novos features. Podem haver várias iterações desse processo até chegar ao modelo final.
